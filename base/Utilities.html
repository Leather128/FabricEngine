1024 = '1 kb'
1536 = '1.5 kb'
1048576 = '2 mb'

Parameters:

Amount of bytes to format and return.

(Optional, Default = false) Whether or not to only format the value of bytes (ex: '1.5 mb' -> '1.5').

precision

(Optional, Default = 2) The precision of the decimal value of bytes. (ex: 1 -> 1.5, 2 -> 1.53, etc).
