class Info

package base

extends TextField

@:directlyUsed

Simple extension of openfl.text.TextField that displays some basic status info about the game (FPS, Memory, etc).

Constructor

Variables

@:value(0.0)current_time:Float = 0.0

Current elapsed program time in milliseconds.

@:value(0)fps:Int = 0

Current frames per second of the program.

@:value(0)memory_usage:Float = 0

Current memory usage of the program in megabytes.

@:value(0)memory_usage_peak:Float = 0

The peak of the memory usage of the program in megabytes.

@:value([])times:Array<Float> = []

Array of all previous times (in milliseconds).
