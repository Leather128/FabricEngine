class Sprite

package base

extends FlxSprite

extended by AlphabetChar, AtlasItem, TrackingSprite

@:directlyUsed

Simple extension to flixel.FlxSprite that is used for easier handling of basic things that should be defaults.

Parameters:

name

Name of the animation.

prefix

Prefix of the animation in the spritesheet.

framerate

Framerate of the animation (FPS).

Whether the animation is looped.
